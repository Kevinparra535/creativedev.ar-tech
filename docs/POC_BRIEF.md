# **AR Immersive Interior Design Platform**

## Intent

Quiero ver si es viable hacer levantamiento de un entorno real y visualizacion del modelo 3D disenado por arquitectos para analizar diseno de interiores y hacer mas visible y entendible para los usuarios como quedaria el espacio que estan intentando remodelar construir usando sus telefonos/ipads con SO iOS, logrando que solo tome maximo dos minutos el usuario hacer esto.

## Hypothesis

Si el arquitecto tiene un modelo 3D del espacio real (casa, apartamento, local, etc) y ha hecho remodelaciones un nuevo diseno de interior etc, el usuario puede ver esto de manera inmersiva usando AR, si el usuario camina dentro de la casa X con su telefono, entonces puede ver el modelo 3D a escala real y ver como cambiaria los espacios y asi no dedicarse a solo renders u otras maneras mas manuales.

## Success Signal

Cuando en menos de dos minutos el usuario (arquitecto) pueda -> seleccionar modelo -> escanear entorno real -> navegar dentro del modelo 3D con su celular donde el AR reemplaza la realidad por el contenido del modelo 3D donde apunte.

## Non-goals

Limitantes en dispositivos, que tecnologias tenemos que usar

## Tooling

Claude, OpenAI, ARKit, RealityKit, Expo, React Native, TypeScript, Swift, Modulos Nativos

## Timebox

2 semanas maximo
